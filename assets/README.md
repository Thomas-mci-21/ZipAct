# Assets

This directory contains images for the README.

## Required Images

Please add the following images from the paper:

1. **zipact_architecture.png** - The ZipAct architecture diagram showing:
   - State Table with Goal State, World State, Constraint State
   - State Updater (U) module
   - Actor (π) module
   - LLM components

2. **context_snowball.png** - The context snowball comparison showing:
   - ReAct: O(T²) quadratic growth
   - ZipAct: O(T) linear growth
   - Visual representation of context length over time

## Image Specifications

- Format: PNG
- Recommended width: 800-1200px
- Background: White or transparent
- Quality: High resolution for GitHub display

## Usage

These images are referenced in `README.md`:
```markdown
<img src="assets/zipact_architecture.png" alt="ZipAct Architecture" width="700">
<img src="assets/context_snowball.png" alt="Context Snowball" width="600">
```
